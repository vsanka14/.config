# Dotfiles

macOS development environment with a consistent Tokyo Night theme.

## Tools

- **Terminal:** WezTerm
- **Editor:** Neovim (AstroNvim)
- **Multiplexer:** Tmux
- **Shell:** Zsh + Oh My Posh
- **Window Manager:** AeroSpace
- **Status Bar:** Sketchybar
- **File Manager:** Yazi
- **Keyboard:** Karabiner-Elements

## Quick Start

```bash
git clone git@github.com:vsanka14/.config.git ~/.config
cd ~/.config
./install.sh
```

See [docs/migration.md](docs/migration.md) for detailed setup guide and troubleshooting.
