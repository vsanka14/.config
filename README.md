# Dotfiles

Personalized macOS development environment built around a keyboard-driven, terminal-centric, single-screen workflow.

## Tools

- **Terminal:** Kitty
- **Editor:** Neovim
- **Multiplexer:** Tmux
- **Shell:** Zsh
- **Tiling Window Manager:** AeroSpace
- **Status Bar:** Sketchybar
- **Key remapper**: Karabiner Elements
- **File Manager:** Yazi
